KEY FEATURES:
=============

✓ FastAPI async REST API with CORS
✓ Celery distributed task processing with Redis broker
✓ PostgreSQL with SQLAlchemy async ORM
✓ Server-Sent Events (SSE) for real-time job updates
✓ 5-stage video processing pipeline:
  1. Download (yt-dlp, 720p max, metadata extraction)
  2. Transcribe (ElevenLabs Scribe v2, diarization, timestamps)
  3. Detect OST (Claude Vision, frame batching, 8 types)
  4. Translate (Claude 3.5 Sonnet, contextual notes, no em-dashes)
  5. Export (python-docx, 4-column table, Lofte Studios format)
✓ Job management with retry capability
✓ Batch job submission
✓ Dashboard statistics (daily/weekly/total, costs, active jobs)
✓ Health monitoring (DB, Redis, workers)
✓ Comprehensive error handling & logging

CONFIGURATION:
==============

Required Environment Variables:
- DATABASE_URL: PostgreSQL connection string
- REDIS_URL: Redis connection URL
- ELEVENLABS_API_KEY: ElevenLabs API key
- ANTHROPIC_API_KEY: Anthropic/Claude API key

Optional (with sensible defaults):
- DEBUG: Enable debug mode (default: false)
- SECRET_KEY: JWT secret (default: dev key)
- STORAGE_PATH: Output directory (default: /data/voxtranslate)
- TEMP_PATH: Temporary files (default: /tmp/voxtranslate)
- CORS_ORIGINS: Allowed CORS origins
- CELERY_WORKER_CONCURRENCY: Worker pool size (default: 4)
- DOWNLOAD_TIMEOUT, TRANSCRIPTION_TIMEOUT, TRANSLATION_TIMEOUT


QUICK START:
============

1. cp .env.example .env
2. Edit .env with your API keys and database credentials
3. docker-compose up -d
4. API: http://localhost:8000
5. Docs: http://localhost:8000/docs
